NER Engine Details

CRF (Conditional Random Fields)

Our statistical model implements a custom feature extraction pipeline that analyzes token-level attributes including:

Prefix/Suffix (1-3)
IsTitle Case Check
Digit Pattern Match
POS Tag Integration
Training Result

Dataset: CoNLL2003 | Method: L-BFGS | F1-Score: 90.32%

Neural Transformer (SpaCy)

Utilizing the `en_core_web_sm` pipeline (v3.0+), this model leverages deep learning embeddings to perform entity extraction. It excel in identifying entities within ambiguous contexts where statistical patterns may fail.
