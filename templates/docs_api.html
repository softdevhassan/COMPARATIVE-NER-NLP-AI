API Reference

The Entify API allows you to programmatically compare CRF and spaCy NER models. It is open to the public for research and development purposes.

Input Sanitization (HTML Stripping) Active

Inference Endpoint

POST /api/compare

Submit text to both CRF and spaCy engines for side-by-side analysis.

Request Headers

Content-Type: application/json

Request Body

{
  "text": "Apple Inc. was founded in Cupertino.",
  "mode": "compare" // Optional: "compare", "crf", or "spacy"
}

Response Schema

CRF Result Object

  • entities Array of detected entity objects
  • processing_time Seconds taken for inference

SpaCy Result Object

  • entities Array of detected entity objects
  • processing_time Seconds taken for inference

Error Codes

400 Bad Request (Empty text provided)
413 Payload Too Large (Over 5000 characters)
500 Internal Server Error (Model inference failed)

Open to the Public

You can call this endpoint from any cross-origin client. CORS is enabled for all domains to support the open research community.
